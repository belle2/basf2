Note for Genfit developers:

the genfit code is mirrrored at KEK. To pull a new genfit version from sourceforge, go to:

https://belle2.cc.kek.jp/internal/cgi-bin/UpdateGenfit.py



to set and commit the new version, type (replace XXXX with the version number which you can see on the URL):

svn propset svn:externals 'code2 -rXXXX https://belle2.cc.kek.jp/genfit/trunk' .
svn ci -m "update genfit revision"
