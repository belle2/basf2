Note for Genfit developers:

to set and commit the new version, type (replace XXXX with the version number which you can see on the URL):

cd $BASF2_DIR/genfit2
svn propset svn:externals 'code2 -rXXXX https://github.com/GenFit/GenFit/trunk' .
svn up  #check if this works, otherwise check revision number, and call the website from above again
svn ci -m "update genfit revision"
