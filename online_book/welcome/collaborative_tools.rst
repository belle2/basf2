.. admonition:: Key points
    :class: key-points

    * `sphinx <https://software.belle2.org/>`_ contains documentation of
      all python functions
    * Questions can be asked at `questions.belle2 <questions.belle2.org/>`_.
    * `Confluence <https://confluence.desy.de/>`_ is our wiki, but it's not that
      reliable for software
    * `Stash <stash.desy.de/>`_ holds our source code
