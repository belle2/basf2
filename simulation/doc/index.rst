Physics list for Geant4
=======================

We have two options for the input physics list for Geant4. The default one is "FTFP_BERT", the popular physics list for high energy physics. The second one is "Belle2PhysicsList", optimized for the Belle II needs. The details on the "Belle2PhysicsList" is explained below.
