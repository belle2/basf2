Import Error

Python has just experienced an import error. The button you have just clicked needs the pillow python module to function properly. It seems that it is missing on this installation of basf2.
To proceed, try installing the module from the terminal with the following set of commands:

source activate
pip install pillow
deactivate

And then try to restart the python webserver. Basf2 environment might also need to be set up again after deactivate-ing. If your python webserver fails to start after installing the module, try setting up the basf2 environment.
After running source activate, your terminal should begin lines with text similar to:

(virtualenv)user@computer-name:~$

If they do not or has the installation failed, you may be experiencing more serious problems. In this case, please consult your computer administrator or try reinstalling basf2.

We apologise for the inconvenience.
